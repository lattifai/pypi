# LattifAI PyPI

Public PyPI index for LattifAI packages.

## Usage

```bash
pip install lattifai-core --extra-index-url https://lattifai.github.io/pypi/simple/
```

Or configure pip globally:

```ini
# ~/.pip/pip.conf
[global]
extra-index-url = https://lattifai.github.io/pypi/simple/
```

## Packages

- **lattifai-core** - Core alignment engine for LattifAI
